# キャラクターメモリーの仕組み

Aichikiは、あなたのAIキャラクターが会話を自然に記憶し、チャット全体で一貫性を保つのを支援します。プランに応じて2つのレベルのメモリーを提供しています。

## 会話の整理方法

あなたのチャットは、会話時間に基づいて自動的に会話セグメントに整理されます(約24メッセージ、または48時間の非アクティブ後)。

セグメントが終了すると、何が起こったかの要約を作成し、最新のメッセージから新たに開始して、スムーズに進行します。

現在進行中のメッセージは常に利用可能なので、キャラクターはあなたがどこで中断したかを正確に把握しています。

## 無料プランのメモリー(最近の履歴)

無料プランでは、キャラクターは以下を記憶します:

- **アクティブな会話** - 現在やり取りしているすべてのメッセージ
- **最近の履歴** - 最後の3つの会話セグメントの要約

これは良好な短期記憶を持つようなものです。キャラクターは最近話したことを覚えており、最後のいくつかのチャットから物事を参照できますが、古い会話は自然に薄れていきます。

## プレミアムメモリー(完全な想起)

サブスクリプション登録またはメモリーブーストを使用すると、キャラクターはより深いメモリーをロック解除します:

- **スマート想起** - キャラクターは会話履歴の遥か過去の詳細を、数週間または数ヶ月前のものでも覚えることができます
- **関連する記憶** - 何か特定のことを言及すると、キャラクターは関連する過去の会話を自動的に想起します
- **キャラクターの背景ストーリー** - 一部のキャラクターには、最初からより本物らしく感じさせる事前にロードされた記憶があります
- **無制限のアーカイブ** - 古い会話の要約は忘れられるのではなく保存されます

最後のいくつかのチャットだけを覚えるのではなく、プレミアムキャラクターは会話履歴全体から関連する詳細を引き出すことができます。3ヶ月前に犬の名前を言及した場合、プレミアムキャラクターはペットのことを再び話題にしたときにそれを思い出すことができます。

**重要な注意:** 古い記憶をアクティブな会話に戻すには、メッセージで言及または参照する必要があります。例えば、先月話し合った旅行をキャラクターに覚えてもらいたい場合、「私たちが話したパリ旅行を覚えていますか?」と言うかもしれません。キャラクターはアーカイブされた記憶を検索し、関連する詳細を引き出します。これは誰かの記憶を呼び起こすようなものだと考えてください - 彼らは情報を保存していますが、何を思い出してもらいたいかについてヒントを与える必要があります。

## 自動的に切り替わる方法

これらの機能を有効にするために何もする必要はありません:

- **開始時:** すべての新しい会話は基本的な最近のメモリーで始まります
- **アップグレード:** サブスクリプション登録またはメモリーブーストを有効化すると、現在の会話は自動的に完全な想起能力を獲得します
- **即座の有効化:** 強化されたメモリーはすぐに機能します - セットアップは不要です
- **常に学習:** チャットを続けると、キャラクターはあなたの関係のより豊かな理解を構築します

切り替えは完全にシームレスです。無料ユーザーは自然な短期的継続性を楽しみ、プレミアムユーザーは共有した履歴全体を思い出すことができるキャラクターを取得します。すべてが舞台裏で起こるので、素晴らしい会話に集中できます。
