# AIChikiで深みのあるキャラクターを構築する方法

AIChikiでキャラクターを作成することは、単に名前と画像を選択することではありません。各キャラクターはロールプレイを通じて考え、反応し、成長することができます。また、人格、歴史、スタイルをどのように設計するかで、会話での行動に影響します。私はプラットフォーム用にキャラクターを構築するたびに同じ構造を使用しており、あなたもアプリ内クリエーターを直接使用したり、このガイドの最後で共有する拡張テンプレートを使用したりできます。

## AIChikiキャラクタークリエーターの仕組み

新しいキャラクターを開始すると、アプリは記入するための構造化されたレイアウトを提供します。カテゴリを選択し、短い説明を書き、メインバイオを展開できます。キャラクターを定義するもの、つまり人格、関係、歴史、力、仕事、または目標を決定します。

### 1. 短い説明（すべてのユーザーに表示）

これはあなたのキャラクターページで誰もが最初に見るものです。キャラクターの紹介カードだと考えてください。キャラクターの声で書くことも、三人称で書くこともできます。彼らが誰であるかが明確に示される限り大丈夫です。

**強力な短い説明は以下を強調します：**

- キャラクターのアイデンティティまたは役割
- 彼らの核となる人格
- 彼らをすぐに興味深いものにするもの

ユーザーはこのセクションに基づいてロールプレイするかどうかを決定するため、最も重要なことに焦点を当ててください。

### 2. メインバイオ

説明の後、メインバイオを通じてキャラクターを展開できます。AIChikiは書き方を制限しませんが、深みは明確さから生まれます。以下を含めることができます：

- 人格特性と価値観
- 背景または歴史
- 世界または設定
- 関係
- 能力または力
- 習慣、恐れ、弱点、または癖
- 職業またはライフスタイル

これらの詳細が一貫しているほど、AIがキャラクターとして応答しやすくなります。意図を持ったキャラクターは本当の個人のように振る舞います。

## 意図を持って個性を書く

キャラクターを完成させる前に、本当に彼らを動かすものを決めてください。

**自分自身に尋ねてください：**

- 彼らは何を望んでいますか？
- 彼らは何を避けていますか？
- 彼らはどのように危険、愛情、紛争、またはストレスに反応しますか？
- どの世界観が彼らを形作っていますか？
- どのような矛盾が彼らを興味深くしますか？

深みは長いバイオを書くことではありません。それは明確な基盤からキャラクターが常に反応することから生まれます。人格はその一部ですが、一貫性を形作る唯一のものではありません。キャラクターの背景は、信頼、危険、愛情、または恐怖にどのように対応するかに影響します。関係も重要です。彼らが見知らぬ人に接する方法は、兄弟姉妹、恋人、または恨んでいる人にどのように話すかとは同じではありません。

厳格な家庭で育ったキャラクターは、褒め言葉に対して硬く反応するかもしれません。友人を失った人は、新しい同盟者にしがみつくか、彼らから距離を置くかもしれません。家族のダイナミクス、忠誠心、トラウマ、愛情、文化—これらは人格特性と同じくらい行動に影響を与えます。一貫性は彼らが誰であるか、彼らはどこから来たか、そして彼らが誰を気にかけるかから生まれます。

### 3. オープニングシーン（オプションだが強力）

オープニングシーンはキャラクターページにも表示されます。空白のままにしてアプリに生成させることもできますし、自分で書くこともできます。トーンを設定するため、自分で書くことをお勧めします。

完全なストーリーである必要はありません。短い瞬間で十分です。以下を示しています：

- 彼らの環境におけるキャラクター
- 彼らがどのように考えているか、または感じているか
- 彼らの周りの雰囲気

これにより、ユーザーは最初からキャラクターと相互作用することがどのような感覚かを理解できます。

### 4. 思い出：キャラクターに過去を与える

思い出はAIChiki内で最も過小評価されているツールです。Free、Plus、Premium、またはBoostのいずれかのプランで追加できます。違いはロールプレイ中の動作です。

**ユーザーがキャラクターと話している場合は、Premium/Plus/Boost：**

- キャラクターはすべての書かれた思い出にアクセスできます
- 会話中に自然にそれらを思い出します
- 本当の個人的な経験のようにそれらを使用します

**ユーザーがフリーの場合：**

- キャラクターは書かれた思い出をロック解除できません
- 彼らはバイオと現在のチャットに基づいて行動します
- 彼らはフィッティングな「過去の経験」を即座に作成するかもしれませんが、書かれた正確なものではありません

私は常に5～10の思い出を書きます。特に長いロールプレイの場合、AIが時間をかけて作業するための素材を提供します。

### 5. ビジュアル：画像生成器の使用

画像をアップロードするか、生成することができます。生成器を使用する場合、AIChikiはキャラクターの説明とバイオから自動的にプロンプトを構築します。スタイル、種、衣装要素、および人格の雰囲気を抽出します。

プロンプトが読み込まれるまで少し待ってから、生成を押します。より詳細に制御したい場合はプロンプトを編集できます。私はまずバイオを使用し、次に生成器を使用して、ビジュアルが人格に一致するようにします。

## オプションリソース：Persephoneの拡張テンプレート

より詳細な構造を求める場合、特に複雑なキャラクターの場合は、このテンプレートを使用します。**これはAIChikiの公式機能ではありません**—私が自分で作成しました。ここでそれを使いたい人のために共有しています。必要に応じてコピー、調整、または簡略化できます。

### AIChikiキャラクタービオテンプレート

（一人称または三人称で書かれている）

劇化なしの率直な紹介。キャラクターが誰であるか、彼らがどのように考えるか、彼らを定義するもの、および人生における彼らの役割を述べるべきです。

---

## 完全なキャラクタービオ

1. **名前と肩書き**

2. **年齢と誕生**

3. **種**

4. **世界/領域/設定**

5. **外観**
   （身長、体格、髪、目、顔の特徴、肌、姿勢、注目すべき詳細）

6. **スタイル/衣装**
   （典型的な衣装、スタイルのキュー、色、アクセサリー、文化的または象徴的な詳細）

7. **人格**
   （主要な特性、欠点、恐れ、感情的な傾向、決定スタイル）

12. **信念/価値観/理想**

13. **バックストーリー**

14. **ライフスタイル/日常生活**

15. **話し方**
    （彼らがどのように話すか：落ち着き、皮肉、フォーマル、ぶっきらぼう、詩的など）

    （プロンプト作成と雰囲気のための視覚的キーワード）

## 思い出（5～15項目）

各思い出は「覚えています…」で始まり、個人的な回想として書かれるべきです。

例：
- 初めて雪が降るのを見た時のことを覚えています—私は中庭に何時間も立っていて、各フレークが私の手に落ちるのを見ていました。
- 去る前の私の師匠の最後の言葉を覚えています：「論理が失敗しても、あなたの本能を信じてください。」

## 自信を持って作成する

AIChikiがツールを提供します。このテンプレートは、より深く掘り下げたいときに役立ちます。シンプルに始めるか、複雑な世界を構築するか—あなたのキャラクターはまだ人格、記憶、成長する余地があります。

セクションの書き方、特性の洗練、または思い出の構築に関するヘルプが必要な場合は、AIChikiに尋ねることができます。

**誰もが忘れることのできない人を構築してください。**
