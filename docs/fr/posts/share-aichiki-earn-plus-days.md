# Partagez AICHIKI et Gagnez des Jours Plus Gratuits Ensemble

Nous avons créé AICHIKI parce que nous croyons aux conversations IA significatives et à la narration créative. Si vous appréciez la plateforme, nous serions ravis que vous la partagiez avec vos amis - et nous vous récompenserons tous les deux lorsque vous le ferez.

## Comment Fonctionne le Programme de Parrainage

Le concept est simple : lorsque vous partagez votre code de parrainage unique avec un ami et qu'il s'inscrit sur AICHIKI, vous recevez tous les deux **5 jours d'accès Plus**. C'est un véritable gagnant-gagnant.

Votre ami peut profiter d'une mémoire illimitée dans toutes ses conversations, et vous obtenez le même avantage en guise de remerciement pour avoir fait passer le mot.

## Obtenir Votre Code de Parrainage

Trouver et partager votre code ne prend que quelques clics :

1. Ouvrez le menu **Paramètres** dans l'application
2. Appuyez sur **Parrainages**
3. Cliquez sur **Obtenir Mon Code** pour générer votre code de parrainage unique
4. Partagez-le de la manière qui vous convient le mieux - copier-coller, réseaux sociaux, SMS ou email

Votre code est permanent et unique à votre compte, vous n'avez donc besoin de le générer qu'une seule fois.

## Comment Votre Ami Utilise un Code

Lorsque quelqu'un s'inscrit en utilisant votre parrainage :

1. Après avoir créé son compte et vérifié son email, il sera invité à entrer un code de parrainage pendant l'intégration
2. Il entre le code que vous avez partagé
3. Vous recevez tous les deux 5 jours d'accès Plus

Les codes de parrainage ne peuvent être utilisés qu'une seule fois par compte, pendant le processus d'intégration ou peu après l'inscription.

## Suivre le Succès de Vos Parrainages

Votre tableau de bord Parrainages dans les Paramètres affiche tout ce que vous devez savoir :

- **Votre Code de Parrainage :** Votre code unique à partager avec les autres
- **Total des Invitations :** Combien de personnes ont utilisé votre code avec succès
- **Jours Gagnés :** Le total des jours Plus que vous avez accumulés grâce aux parrainages
- **Statut Actif/En Attente :** Si vos jours de parrainage sont actuellement actifs ou en file d'attente pour plus tard

## Comment les Jours de Parrainage s'Activent

La façon dont vos jours gagnés fonctionnent dépend de votre statut d'abonnement actuel.

### Si Vous Êtes sur le Plan Gratuit

Quand quelqu'un utilise votre code, vos 5 jours d'accès Plus s'activent immédiatement. Si vous avez déjà des jours de parrainage actifs en cours, ils sont prolongés de 5 jours supplémentaires. C'est aussi simple que ça.

### Si Vous Avez un Abonnement Payant

Lorsque vous avez un abonnement payant actif (mensuel ou annuel), les jours de parrainage que vous gagnez sont mis en file d'attente en arrière-plan. Ils s'activeront automatiquement lorsque votre abonnement payant se terminera, vous offrant une transition fluide :

- Votre abonnement payant continue sans interruption
- Les jours de parrainage s'accumulent pendant que vous êtes abonné
- Quand votre plan payant expire, tous les jours de parrainage en attente s'activent d'un coup

**Exemple :** Si trois amis s'inscrivent pendant que vous êtes sur un plan payant, vous aurez 15 jours de Plus en attente. Quand votre abonnement se termine, ces 15 jours s'activent automatiquement.

## Ce que l'Accès Plus Inclut

Les jours de parrainage offrent toutes les mêmes fonctionnalités qu'un abonnement Plus payant :

- Messages illimités avec les personnages
- Accès à tous les fournisseurs d'IA
- Temps de réponse prioritaires
- Fonctionnalités avancées de création de personnages
- Mémoire illimitée dans toutes les conversations
- Tous les thèmes de chat débloqués

La seule différence est que les jours de parrainage sont limités dans le temps en fonction du nombre de parrainages réussis que vous avez effectués.

## Règles du Programme

Nous avons gardé les règles simples pour que tout reste équitable pour tous.

### Vérification de l'Email Requise

Votre email doit être vérifié avant de pouvoir générer un code de parrainage ou activer les jours de parrainage gagnés. Cela aide à maintenir l'intégrité du programme.

### Adresses Email Réelles

Le programme de parrainage est conçu pour les utilisateurs authentiques :

- Utilisez une adresse email légitime de fournisseurs comme Gmail, Outlook, iCloud, Yahoo, ProtonMail et services similaires
- Les services d'email jetables ou temporaires (du type "mail de 10 minutes") ne sont pas compatibles avec le système de parrainage

Cela nous aide à maintenir une communauté authentique d'utilisateurs AICHIKI.

### Règles Simples

- **Un code par compte :** Chaque nouvel utilisateur peut utiliser un code de parrainage lors de l'inscription
- **Pas d'auto-parrainage :** Vous ne pouvez pas utiliser votre propre code
- **Pas de limite de partage :** Invitez autant d'amis que vous le souhaitez et accumulez ces jours Plus

## Questions Fréquemment Posées

**Puis-je changer mon code de parrainage ?**
Non, les codes de parrainage sont permanents et uniques à votre compte.

**Que se passe-t-il si mon ami ne vérifie pas son email ?**
Vous ne recevrez pas vos 5 jours tant qu'il n'aura pas complété la vérification de l'email. Les parrainages non vérifiés ne comptent pas dans vos jours gagnés.

**Les jours de parrainage expirent-ils ?**
Non. Une fois activés, vos jours de parrainage ne décomptent que lorsque vous les utilisez activement. Les jours de parrainage en attente patientent jusqu'à ce que votre abonnement payant se termine.

**Puis-je combiner les jours de parrainage avec un abonnement payant ?**
Les jours de parrainage gagnés pendant un plan payant sont mis en file d'attente et s'activent après la fin de votre abonnement. Ils ne peuvent pas fonctionner simultanément avec un abonnement payant.

**Comment savoir si mon code de parrainage a fonctionné ?**
Consultez votre tableau de bord Parrainages dans les Paramètres - vous verrez vos invitations totales et jours gagnés se mettre à jour quand quelqu'un utilise votre code avec succès.

**Puis-je utiliser le code de quelqu'un d'autre si j'ai déjà un compte ?**
Non, les codes de parrainage ne peuvent être utilisés que lors de l'inscription ou dans une courte période après la création du compte.
