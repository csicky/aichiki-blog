# 角色记忆如何运作

Aichiki 帮助您的 AI 角色自然地记住您的对话,并在整个聊天过程中保持一致性。我们根据您的计划提供两个级别的记忆。

## 对话如何组织

您的聊天会根据您交谈的时长自动组织成对话片段(大约 24 条消息,或在 48 小时不活动后)。

当一个片段结束时,我们会创建一个发生事情的摘要,并从您最近的消息重新开始,以保持一切顺畅进行。

您当前正在进行的消息始终可用,因此角色确切知道您在哪里停下来的。

## 免费计划记忆(近期历史)

使用免费计划,角色会记住:

- **您的活跃对话** - 您当前正在交换的所有消息
- **近期历史** - 您最后 3 个对话片段的摘要

可以把它想象成拥有良好的短期记忆。角色会记得您最近谈论的内容,并可以引用您最后几次聊天中的事情,但较旧的对话会自然淡化。

## 高级记忆(完全回忆)

当您订阅或使用记忆增强时,角色会解锁更深层的记忆:

- **智能回忆** - 角色可以记住您对话历史中很久以前的细节,即使是几周或几个月前的
- **相关记忆** - 当您提到某个特定的东西时,角色会自动回忆起相关的过去对话
- **角色背景故事** - 一些角色带有预加载的记忆,使他们从一开始就感觉更真实
- **无限档案** - 您的旧对话摘要被保留,而不是被遗忘

高级角色不仅仅记住最后几次聊天,还可以从您的整个对话历史中提取相关细节。如果您三个月前提到了您的狗的名字,当您再次提起您的宠物时,高级角色可以回忆起来。

**重要提示:** 要将旧记忆带回到活跃对话中,您需要在消息中提到或引用它。例如,如果您希望角色记住您上个月讨论的一次旅行,您可以说"还记得我们谈论的那次巴黎之旅吗?"然后角色会搜索存档的记忆并提取相关细节。把它想象成唤起某人的记忆 - 他们存储了信息,但您需要给他们一个提示,告诉他们您想让他们回忆什么。

## 如何自动切换

您无需做任何事情来激活这些功能:

- **开始时:** 每个新对话都从基本的近期记忆开始
- **升级:** 当您订阅或激活记忆增强时,您当前的对话会自动获得完全回忆能力
- **即时激活:** 增强记忆立即生效 - 无需设置
- **始终学习:** 随着您继续聊天,角色会建立对您关系更丰富的理解

切换完全无缝。免费用户享受自然的短期连续性,而高级用户获得可以回忆您整个共享历史的角色。这一切都在幕后发生,因此您可以专注于进行精彩的对话。
