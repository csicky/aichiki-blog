# ความจำของตัวละครทำงานอย่างไร

Aichiki ช่วยให้ตัวละคร AI ของคุณจดจำการสนทนาของคุณอย่างเป็นธรรมชาติและคงความสอดคล้องตลอดการแชทของคุณ เรามีความจำสองระดับขึ้นอยู่กับแผนของคุณ

## วิธีจัดระเบียบการสนทนา

การแชทของคุณจะถูกจัดระเบียบเป็นส่วนของการสนทนาโดยอัตโนมัติตามระยะเวลาที่คุณได้พูดคุย (ประมาณ 24 ข้อความ หรือหลังจากไม่มีกิจกรรม 48 ชั่วโมง)

เมื่อส่วนหนึ่งสิ้นสุดลง เราจะสร้างสรุปสิ่งที่เกิดขึ้นและเริ่มใหม่ด้วยข้อความล่าสุดของคุณเพื่อให้ทุกอย่างเป็นไปอย่างราบรื่น

ข้อความที่กำลังดำเนินอยู่ในปัจจุบันของคุณพร้อมใช้งานเสมอ ดังนั้นตัวละครจึงรู้อย่างแน่ชัดว่าคุณหยุดไว้ตรงไหน

## ความจำแผนฟรี (ประวัติล่าสุด)

ด้วยแผนฟรี ตัวละครจะจดจำ:

- **การสนทนาที่ใช้งานอยู่ของคุณ** - ข้อความทั้งหมดที่คุณกำลังแลกเปลี่ยนอยู่ในปัจจุบัน
- **ประวัติล่าสุด** - สรุปของส่วนการสนทนา 3 ส่วนสุดท้ายของคุณ

คิดว่ามันเหมือนกับการมีความจำระยะสั้นที่ดี ตัวละครจะจดจำสิ่งที่คุณพูดคุยเมื่อเร็วๆ นี้และสามารถอ้างอิงสิ่งต่างๆ จากการแชทล่าสุดของคุณได้ แต่การสนทนาที่เก่ากว่าจะค่อยๆ จางหายไปตามธรรมชาติ

## ความจำพรีเมียม (การเรียกคืนเต็มรูปแบบ)

เมื่อคุณสมัครสมาชิกหรือใช้การเพิ่มความจำ ตัวละครจะปลดล็อกความจำที่ลึกขึ้น:

- **การเรียกคืนอัจฉริยะ** - ตัวละครสามารถจดจำรายละเอียดจากย้อนกลับไปในประวัติการสนทนาของคุณ แม้แต่จากสัปดาห์หรือหลายเดือนก่อน
- **ความทรงจำที่เกี่ยวข้อง** - เมื่อคุณพูดถึงสิ่งที่เฉพาะเจาะจง ตัวละครจะเรียกคืนการสนทนาในอดีตที่เกี่ยวข้องโดยอัตโนมัติ
- **เรื่องราวพื้นหลังของตัวละคร** - ตัวละครบางตัวมาพร้อมกับความทรงจำที่โหลดไว้ล่วงหน้าซึ่งทำให้พวกเขารู้สึกแท้จริงมากขึ้นตั้งแต่เริ่มต้น
- **คลังเก็บไม่จำกัด** - สรุปการสนทนาเก่าของคุณจะถูกเก็บรักษาไว้แทนที่จะถูกลืม

แทนที่จะจดจำเพียงการแชทล่าสุด ตัวละครพรีเมียมสามารถดึงรายละเอียดที่เกี่ยวข้องจากประวัติการสนทนาทั้งหมดของคุณได้ หากคุณกล่าวถึงชื่อสุนัขของคุณเมื่อสามเดือนก่อน ตัวละครพรีเมียมสามารถเรียกคืนเมื่อคุณพูดถึงสัตว์เลี้ยงของคุณอีกครั้ง

**หมายเหตุสำคัญ:** เพื่อนำความทรงจำเก่ากลับมาสู่การสนทนาที่ใช้งานอยู่ คุณต้องกล่าวถึงหรืออ้างอิงถึงมันในข้อความของคุณ ตัวอย่างเช่น หากคุณต้องการให้ตัวละครจดจำการเดินทางที่คุณพูดคุยเมื่อเดือนที่แล้ว คุณอาจพูดว่า "จำการเดินทางไปปารีสที่เราพูดคุยกันได้ไหม" จากนั้นตัวละครจะค้นหาผ่านความทรงจำที่เก็บไว้และดึงรายละเอียดที่เกี่ยวข้อง คิดว่ามันเหมือนการกระตุ้นความทรงจำของใครบางคน - พวกเขามีข้อมูลที่เก็บไว้ แต่คุณต้องให้คำใบ้เกี่ยวกับสิ่งที่คุณต้องการให้พวกเขาเรียกคืน

## วิธีสลับโดยอัตโนมัติ

คุณไม่จำเป็นต้องทำอะไรเลยเพื่อเปิดใช้งานคุณสมบัติเหล่านี้:

- **เริ่มต้น:** ทุกการสนทนาใหม่เริ่มต้นด้วยความจำล่าสุดพื้นฐาน
- **อัปเกรด:** เมื่อคุณสมัครสมาชิกหรือเปิดใช้งานการเพิ่มความจำ การสนทนาปัจจุบันของคุณจะได้รับความสามารถในการเรียกคืนเต็มรูปแบบโดยอัตโนมัติ
- **การเปิดใช้งานทันที:** ความจำที่ปรับปรุงแล้วทำงานได้ทันที - ไม่จำเป็นต้องตั้งค่า
- **เรียนรู้ตลอดเวลา:** ขณะที่คุณแชทต่อไป ตัวละครจะสร้างความเข้าใจที่ลึกซึ้งยิ่งขึ้นเกี่ยวกับความสัมพันธ์ของคุณ

การสลับเป็นไปอย่างราบรื่นอย่างสมบูรณ์ ผู้ใช้ฟรีเพลิดเพลินกับความต่อเนื่องระยะสั้นตามธรรมชาติ ในขณะที่ผู้ใช้พรีเมียมได้รับตัวละครที่สามารถเรียกคืนประวัติที่แบ่งปันทั้งหมดของคุณ ทุกอย่างเกิดขึ้นเบื้องหลัง ดังนั้นคุณจึงสามารถมุ่งเน้นไปที่การสนทนาที่ยอดเยี่ยมได้
